# RemoteDevAI Web Dashboard

Next.js web application for monitoring and managing AI agents.

## Features

- Project dashboard
- Agent configuration
- Real-time logs and monitoring
- Analytics and insights
- User management

## Setup

```bash
npm install
npm run dev
```

## Building

```bash
# Build for production
npm run build

# Start production server
npm run start
```
