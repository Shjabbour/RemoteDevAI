# Swagger Annotations for Route Files

This document contains all the Swagger/OpenAPI annotations that should be added to each route file.

## Auth Routes (`src/routes/auth.routes.ts`)

### Register Endpoint
```typescript
/**
 * @swagger
 * /api/auth/register:
 *   post:
 *     summary: Register a new user account
 *     description: Create a new user account with email and password. Returns JWT tokens for authentication.
 *     tags: [Authentication]
 *     requestBody:
 *       required: true
 *       content:
 *         application/json:
 *           schema:
 *             $ref: '#/components/schemas/RegisterRequest'
 *     responses:
 *       201:
 *         description: Registration successful
 *         content:
 *           application/json:
 *             schema:
 *               $ref: '#/components/schemas/RegisterResponse'
 *       400:
 *         $ref: '#/components/responses/ValidationError'
 *       429:
 *         $ref: '#/components/responses/RateLimitError'
 */
```

## Projects Routes (`src/routes/projects.routes.ts`)

### List Projects
```typescript
/**
 * @swagger
 * /api/projects:
 *   get:
 *     summary: List all projects
 *     description: Get all projects for the authenticated user with pagination support.
 *     tags: [Projects]
 *     security:
 *       - bearerAuth: []
 *     parameters:
 *       - $ref: '#/components/parameters/PageParam'
 *       - $ref: '#/components/parameters/LimitParam'
 *       - name: includeArchived
 *         in: query
 *         description: Include archived projects
 *         schema:
 *           type: boolean
 *           default: false
 *     responses:
 *       200:
 *         description: Projects retrieved successfully
 *         content:
 *           application/json:
 *             schema:
 *               $ref: '#/components/schemas/ProjectList'
 *       401:
 *         $ref: '#/components/responses/UnauthorizedError'
 *       500:
 *         $ref: '#/components/responses/InternalServerError'
 */
```
