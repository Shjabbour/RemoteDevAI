# RemoteDevAI Agents

Collection of 10 specialized AI agents for different development tasks.

## Agents

1. Coder Agent - Code generation and implementation
2. Debugger Agent - Bug identification and fixes
3. Tester Agent - Test writing and execution
4. Reviewer Agent - Code review and quality checks
5. DevOps Agent - Deployment and infrastructure
6. Designer Agent - UI/UX and styling
7. Writer Agent - Documentation and comments
8. Analyst Agent - Performance analysis
9. Security Agent - Security audits
10. Manager Agent - Project coordination
