# RemoteDevAI MCP Tools

Model Context Protocol tool implementations for AI-environment interaction.

## Tools

- File Operations - Read, write, search files
- Git Operations - Commit, push, pull, branch management
- Terminal - Execute shell commands
- Search - Code and file search
- Database - Query and migration tools

## Usage

These tools are automatically available to AI agents through the MCP protocol.
